## Installing:
### Windows:
I don't fucking know, lmao. Figure it out yourself, I don't use bloat. Find it in youtube or use google or something.

### Linux:
**One line install** (reccoemnded!)
```bash
curl -fsSL https://raw.githubusercontent.com/daveberrys/SendYourCLI/main/install-script/linux.sh | sh
```

**Manual install**

Download the binary from the zip file, extract it, place `SendYourCLI-Linux` in `~/.local/bin`. If you want it to be system-wide, place it in `/usr/bin`. Though, user-only would be better.
